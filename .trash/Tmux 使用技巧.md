### ⚠️ 核心前缀键 (Prefix)

所有 Tmux 的快捷键操作，都必须先按下前缀键。
默认前缀是：**`Ctrl + a`**

---

### 1. 会话管理 (Session)

_相当于游戏的“存档/读档”，这是 Tmux 最核心的功能。_

| 场景     | 终端命令 (Shell)            | 快捷键 (Tmux 内) | 说明                                        |
| -------- | --------------------------- | ---------------- | ------------------------------------------- |
| **新建** | `tmux new -s <名>`          | -                | 新建一个带名字的会话 (推荐)                 |
| **离开** | -                           | `Prefix` + `d`   | **Detach** (挂起)，回到普通终端，程序继续跑 |
| **回来** | `tmux attach -t <名>`       | -                | **Attach** (恢复)，回到之前的会话           |
| **列出** | `tmux ls`                   | `Prefix` + `s`   | 查看所有后台运行的会话                      |
| **改名** | -                           | `Prefix` + `$`   | 重命名当前会话 (方便管理)                   |
| **关闭** | `tmux kill-session -t <名>` | -                | 彻底杀死某个会话                            |

---

### 2. 窗口管理 (Window)

_相当于浏览器的“标签页”，用于区分不同的任务（如：一个跑后端，一个跑前端）。_

| 功能     | 快捷键 (先按 Ctrl+a)    | 说明                                           |
| -------- | ----------------------- | ---------------------------------------------- |
| **新建** | `c`                     | Create，新建一个空白窗口                       |
| **关闭** | `&` (或直接输入 `exit`) | 关闭当前窗口 (会有确认提示)                    |
| **切换** | `n` / `p`               | Next / Previous，切换 下一个 / 上一个 窗口     |
| **指定** | `0` ~ `9`               | 直接跳转到第 N 号窗口                          |
| **列表** | `w`                     | **重要**：显示所有窗口列表，可上下选择回车进入 |
| **改名** | `,`                     | 重命名当前窗口 (如改为 "Backend")              |

---

### 3. 面板管理 (Pane)

_相当于“分屏”，用于在一个视野内对比信息或同时监控。_

| 功能         | 快捷键 (先按 Ctrl+a) | 说明                                              |
| ------------ | -------------------- | ------------------------------------------------- |
| **左右切**   | `%` (Shift + 5)      | 屏幕竖着切一刀，分左右                            |
| **上下切**   | `"` (Shift + ')      | 屏幕横着切一刀，分上下                            |
| **移动**     | `方向键`             | 在不同块之间移动光标                              |
| **最大化**   | **`z`**              | **Zoom**，让当前小块全屏 (再按一次还原)，非常实用 |
| **关闭**     | `x` (或输入 `exit`)  | 关闭当前小块                                      |
| **调整大小** | `Ctrl` + `方向键`    | (按住 Ctrl 不放) 调整分屏线的位置                 |
| **显示编号** | `q`                  | 显示面板编号（按数字键可快速跳转）                |

---

### 4. 滚动模式 (Scroll Mode)

_Tmux 默认不能直接用鼠标滚轮往回看日志，这是新手最大的痛点。_

-   **进入查看模式**：按下 **`Prefix` + `[**`
-   **如何操作**：现在你可以用 `方向键`、`PageUp/Down` 甚至 `Ctrl+u/d` 来上下翻阅历史输出。
-   **退出模式**：按下 **`q`** 即可回到正常的输入模式。

---

### 💡 进阶建议：开启鼠标支持 (强烈推荐)

默认 Tmux 不支持鼠标点选，这很反直觉。你可以通过建立一个配置文件来开启鼠标支持，这样你就可以**直接用鼠标点击切换窗口、拖拽调整分屏大小**了。

**操作步骤：**

1. 在你的用户主目录 (`~`) 下创建一个名为 `.tmux.conf` 的文件：

```bash
nano ~/.tmux.conf

```

2. 在里面粘贴这一行代码：

```text
set -g mouse on

```

3. 保存并退出 (`Ctrl+O` -> Enter -> `Ctrl+X`)。
4. 让配置生效：在终端输入：

```bash
tmux source-file ~/.tmux.conf

```

**搞定！** 现在你可以在 Tmux 里用鼠标点点点了，体验会从“硬核”变得“丝滑”。
