# Chapter 1

## 什么是 Kernel 和 Process？

教材中这段话说的很清晰易懂，摘录下来：

>"xv6 采用传统形式 kernel，一个为正在运行的程序提供服务的特殊程序。每个正在运行的程序，称为 process，具有包含指令、数据和堆栈的内存。这些指令实现程序的计算。数据是计算所作用的变量。堆栈组织程序的过程调用。给定的计算机通常具有许多进程，但只有一个内核。当进程需要调用内核服务时，它会调用 system call，这是操作系统接口中的调用之一。系统调用进入内核；内核执行服务并返回。因此，一个进程交替执行 user space 和 kernel space。"

值得一提的是，shell 也是一个普通的 process，存在于 user space，而不是 kernel 的一部分。
